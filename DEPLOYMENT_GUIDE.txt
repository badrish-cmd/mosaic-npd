# ── Mosaic NPD — Deployment Guide ──────────────────────────────────
#
# Frontend → Vercel  (free tier)
# Backend  → Render  (free tier)
# Database → You already have a PostgreSQL database (keep using it)
#
# ════════════════════════════════════════════════════════════════════
# STEP 1: PUSH TO GITHUB
# ════════════════════════════════════════════════════════════════════
#
# Make sure your project is pushed to a GitHub repository.
# If not already done:
#
#   cd d:\mosaic-product-strategy-engine
#   git init
#   git add .
#   git commit -m "initial commit"
#   git remote add origin https://github.com/YOUR_USERNAME/mosaic-npd.git
#   git push -u origin main
#
#
# ════════════════════════════════════════════════════════════════════
# STEP 2: DEPLOY BACKEND ON RENDER
# ════════════════════════════════════════════════════════════════════
#
# 1. Go to https://render.com and sign up / log in
#
# 2. Click "New +" → "Web Service"
#
# 3. Connect your GitHub repo
#
# 4. Configure:
#      Name:           mosaic-npd-backend
#      Region:         Oregon (or closest to you)
#      Root Directory: backend
#      Runtime:        Python 3
#      Build Command:  pip install -r requirements.txt
#      Start Command:  uvicorn app.main:app --host 0.0.0.0 --port $PORT
#
# 5. Add Environment Variables (click "Advanced" → "Add Environment Variable"):
#      DATABASE_URL  =  your_postgresql_connection_string
#      CLAUDE_API_KEY = (optional, leave blank if not using AI)
#
# 6. Click "Create Web Service"
#
# 7. Wait for it to build and deploy (takes 2-5 minutes)
#
# 8. Copy the URL Render gives you, e.g.:
#      https://mosaic-npd-backend.onrender.com
#
#
# ════════════════════════════════════════════════════════════════════
# STEP 3: DEPLOY FRONTEND ON VERCEL
# ════════════════════════════════════════════════════════════════════
#
# 1. Go to https://vercel.com and sign up / log in
#
# 2. Click "Add New" → "Project"
#
# 3. Import your GitHub repo
#
# 4. Configure:
#      Framework Preset: Create React App
#      Root Directory:   frontend
#      Build Command:    npm run build
#      Output Directory: build
#
# 5. Add Environment Variable:
#      REACT_APP_API_URL = https://mosaic-npd-backend.onrender.com
#      (use the Render URL from Step 2.8 — NO trailing slash)
#
# 6. Click "Deploy"
#
# 7. Wait for build (takes 1-2 minutes)
#
# 8. Your app is live! Vercel gives you a URL like:
#      https://mosaic-npd.vercel.app
#
#
# ════════════════════════════════════════════════════════════════════
# STEP 4: VERIFY
# ════════════════════════════════════════════════════════════════════
#
# 1. Open your Vercel URL in a browser
# 2. Log in with any email/password
# 3. Go to Upload page → upload sample CSVs
# 4. Go to Dashboard → click Refresh
# 5. You should see complaint analysis, opportunities, and concepts
#
#
# ════════════════════════════════════════════════════════════════════
# TROUBLESHOOTING
# ════════════════════════════════════════════════════════════════════
#
# PROBLEM: Frontend shows "Network Error" or can't fetch data
# FIX: Check that REACT_APP_API_URL in Vercel matches your Render URL
#      exactly (no trailing slash). Redeploy frontend after changing.
#
# PROBLEM: Render deployment fails with "database" error
# FIX: Make sure DATABASE_URL env var is set correctly in Render.
#      Format: postgresql://user:password@host:5432/dbname?sslmode=require
#
# PROBLEM: Render shows "build failed"
# FIX: Make sure Root Directory is set to "backend" (not the repo root).
#
# PROBLEM: Vercel shows "build failed"
# FIX: Make sure Root Directory is set to "frontend" and Framework is
#      "Create React App".
#
# PROBLEM: Render free tier sleeps after 15 min of inactivity
# FIX: This is normal on free tier. First request after sleep takes
#      ~30 seconds to wake up. Upgrade to paid ($7/month) for always-on.
#
# PROBLEM: CORS error in browser console
# FIX: The backend already allows all origins ("*"). If you want to
#      restrict it, update allow_origins in main.py to your Vercel URL.
#
#
# ════════════════════════════════════════════════════════════════════
# NOTES
# ════════════════════════════════════════════════════════════════════
#
# - Render free tier: 750 hours/month, sleeps after 15 min idle
# - Vercel free tier: unlimited for personal projects
# - Your PostgreSQL database stays where it is (no change needed)
# - The backend auto-creates tables on startup
# - Environment variables are set in the Render/Vercel dashboards,
#   NOT in .env files (those are only for local development)
